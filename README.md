# 💰 CryptoAdvisorBot

### 🚀 Your First AI-Powered Financial Sidekick!

CryptoAdvisorBot is a beginner-friendly, rule-based chatbot built in Python that helps users make basic cryptocurrency investment decisions. It gives advice based on **profitability** (price trends) and **sustainability** (energy use and long-term viability).

---

## 🎯 Features

- 📈 Analyzes predefined crypto data (Bitcoin, Ethereum, Cardano)
- 🌱 Recommends sustainable coins based on energy efficiency
- 🧠 Rule-based logic using simple `if-else` conditions
- 💬 Interactive chatbot experience inside Jupyter Notebook

---

## 📊 Crypto Dataset

The bot uses a sample dataset with the following info for each coin:

- Price trend (e.g., rising, stable)
- Market capitalization (e.g., high, medium)
- Energy use (e.g., high, medium, low)
- Sustainability score (out of 10)

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Platform:** Jupyter Notebook
- **Libraries:** None (pure Python logic for learning)

---

## 📚 How to Use

1. Clone or download this repository.
2. Open `CryptoAdvisorBot.ipynb` using Jupyter Notebook.
3. Run the setup cell to define the bot and dataset.
4. Use the input cell to ask questions like:
   - `"Which crypto is trending up?"`
   - `"What’s the most sustainable coin?"`
   - `"Tell me about Cardano."`
